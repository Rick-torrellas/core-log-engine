Clase: AppLogger

Herencia: Hereda de Logger

La clase AppLogger es una implementación de puerto (Logger port) que utiliza la librería estándar logging de Python para gestionar la salida de registros de la aplicación.

Constructor

__init__(self, name: str) -> None

Inicializa una nueva instancia del logger.

  • Parámetros:
    • name (str): El nombre del logger, usualmente utilizado para identificar el origen de los mensajes.
  • Descripción: Crea una instancia interna de logging.Logger.

Métodos Públicos

info(self, message: str) -> None

Registra un mensaje con nivel de severidad INFO.

  • message: El contenido del texto a registrar.

debug(self, message: str) -> None

Registra un mensaje con nivel de severidad DEBUG.

  • message: Detalles técnicos útiles para la depuración.
